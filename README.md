context, signals, and the talking points I need before I reach out to a prospect.
Built in n8n, with Tavily for live web research and OpenRouter as the reasoning layer.
It turns an afternoon of manual digging into a few minutes.

Repo: [account-research-automation](https://github.com/clementine-samsoen/account-research-automation)
